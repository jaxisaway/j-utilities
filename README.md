# j-utilities
### a discord utility bot for your favourite group chats.

features:
> gif auto-deletion for users with a certain role (its a crime to be unfunny)

> role adding and removal

> message clearing

> starboards

> birthday messages

> tracks words and phrases

> show who puts things on someone's soul

> massive configuration and logging

> json and database support

## run from terminal

1. install stuff: `pip install -r requirements.txt`
2. put your token in main.py
3. run `python3 main.py`
